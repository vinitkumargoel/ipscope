API Documentation

REST JSON API for IP geolocation. No API key required.

Base URL: https://ipscope.app (or your self-hosted instance). All responses are JSON. Rate limit: 60 requests/minute per IP for bulk endpoints.

GET/api/health

Health check and database status.

{"ok": true, "database": true}

GET/api/me

Returns geolocation for the requesting client's IP. Includes IPv4/IPv6 detection, proxy chain, and HTTPS status.

curl https://ipscope.app/api/me

GET/api/lookup/:ip

Lookup any public IPv4 or IPv6 address.

curl https://ipscope.app/api/lookup/8.8.8.8

Response fields include: ip, country, city, latitude, longitude, timezone, isp, asn, hostname, rdapName, abuseEmail, riskLabel, and more.

POST/api/bulk

Bulk lookup up to 20 IPs. Body: {"ips": ["8.8.8.8", "1.1.1.1"]}

curl -X POST https://ipscope.app/api/bulk \
  -H "Content-Type: application/json" \
  -d '{"ips":["8.8.8.8","1.1.1.1"]}'

Error responses

{"error": "Invalid IP address"}  // HTTP 400
{"error": "Too many IPs (max 20)"}  // HTTP 400
