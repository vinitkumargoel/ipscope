About IPScope

Self-hosted IP geolocation — private, fast, and offline-first.

What is IPScope?

IPScope is a web application that shows your public IP address and geolocation details — country, city, ISP, ASN, timezone, coordinates, and more. Unlike cloud geolocation APIs, lookups run entirely on your server using offline MaxMind-compatible MMDB databases.

Why we built it

  • Privacy: No third-party API calls for core geolocation.
  • Cost: No per-request API fees after database download.
  • Speed: Local MMDB lookups are sub-millisecond.
  • Completeness: IPv4, IPv6, RDAP registry data, reverse DNS, and browser fingerprint tiles.

Technology

Node.js + Express backend, MaxMind MMDB readers (DB-IP + GeoLite2 merge), RDAP for registry enrichment, Leaflet maps, and a responsive bento-grid UI.

Open source

IPScope is open source. Deploy your own instance, customize it, and contribute improvements on GitHub.

Data sources

  • DB-IP City & ASN databases
  • GeoLite2 City (sapics/ip-location-db)
  • Public RDAP registries (RIPE, ARIN, APNIC, etc.)
  • OpenStreetMap tiles
